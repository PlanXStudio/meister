# 인터넷 환경에서 로봇 원격 제어
주소패밀리가 인터넷이고, 프로토콜은 UDP(Datagram)를 사용하는 소켓 객체를 만들어 로봇 원격 제어

## 서버(로봇)와 클라이언트(PC) 구조
- 서버의 포트 번호는 12345
```python
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.bind( ("", 12345) )
```

## 서버는 제어 명령을 수신한 후 해당 명령 수행
소켓객체의 recvfrom()으로 데이터 읽은 후 바이트 배열을 차례로 파이썬 문자열, 소문자, 리스트로 변환
```python
data, client_addr = sock.recvfrom(1024)
data = data.decode().lower().split()
```
